# GnuPG Build

## English

Automated daily build of GnuPG via GitHub Actions, fetching the stable from [GnuPG Download](https://www.gnupg.org/download/index.html) and creating a Release upon success.

Support:
- Linux amd64/arm64
- Windows x64

## 中文

每天自动从 [GnuPG 官网](https://www.gnupg.org/) 获取最新稳定版源码包，然后使用 GitHub Actions 构建并发布 Release。所有流程全部自动化实现

支持平台：

- Linux amd64/arm64
- Windows x64

下载构建产物： [Releases](https://github.com/sweetsky123/gnupg-build/releases)
